# Setup

1. ## clone repository

   ```sh
   git clone https://github.com/dding-g/rewrite-the-stars.git
   cd rewrite-the-stars
   bun i
   ```

2. fill the env

   - github token using for
     - gist create, read
     - github repository read

   ```sh
   GITHUB_TOKEN=***
   GEMINI_API_KEY=***
   ```

3. start
   1.
   ```sh
   bun start
   ```
